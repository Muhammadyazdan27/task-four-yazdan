# task-four-yazdan

# Full Stack System - Complete Integration

# Description
An end-to-end integrated application connecting a responsive HTML/CSS frontend to an MS SQL database via a Node.js backend API. It utilizes asynchronous fetch requests and CORS for seamless, dynamic UI updates without page reloads.

# How to Run
1. Navigate to the `Backend` folder, run `npm install cors express mssql msnodesqlv8`, and start the server with `node server.js`.
2. Navigate to the `Frontend` folder, right-click `index.html`, and select 'Open with Live Server'.
3. Interact with the UI form to send data directly to the database and see dynamic updates.
